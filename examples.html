Projects using iTerm2 tools¶

Some example projects using iterm2-tools. If you know of another project using this, please let me know.

catimg¶

Display an image of a cat from Imgur in your terminal. https://github.com/asmeurer/catimg

Install with

pip install catimg

giphycat¶

Display a gif from Giphy in your terminal.

Install with

pip install giphycat
